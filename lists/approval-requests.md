## リクエストフォーム

新しいリクエストを作成する際は以下の形式で追加：

```
### #XXX - [タイトル]
- **要求日時**: YYYY-MM-DD
- **要求者**: [ユーザー名]
- **操作内容**: [詳細]
- **承認者**: [承認したマスター名]
- **状態**: [保留中/承認済み/拒否/実行済み]
- **実行方法**: [具体的な手順]
```

## 承認ルール

1. **マスター1名の承認で実行可能**
   - パッケージインストール（apt-get, npm, pip）
   - 設定ファイルの変更

2. **全マスターの承認が必要**
   - MASTERS.md の変更
   - 新規マスターの追加
   - セキュリティ関連の重大な変更

3. **自動拒否**
   - 権限なしで設定変更を試みるユーザー
   - danger-listに登録されているユーザー
