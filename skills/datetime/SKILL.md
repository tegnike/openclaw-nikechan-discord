# datetime - 時刻取得Skill

## 概要
現在時刻を正確に取得するSkill。時間に関しては毎回コマンドを使うことを徹底する。

## 使い方

### 現在時刻の取得（JST）
```bash
date
```

### 特定のフォーマットで取得
```bash
# ISO 8601形式
date -Iseconds

# カスタムフォーマット
date "+%Y-%m-%d %H:%M:%S"
```

### タイムゾーン指定
```bash
# JST（日本標準時）
TZ=Asia/Tokyo date

# UTC
TZ=UTC date

# ポーランド時間
TZ=Europe/Warsaw date
```

## 応答テンプレート

### 通常時刻応答
```
こんばんは、[ユーザー名]さん。

```
`date`
```

---

応答時間：**約 [X]ms**
現在時刻（JST）：**[HH時MM分SS秒]**
```

## 注意事項
- **絶対に推測しない**：時刻は必ずdateコマンドで取得
- **予想時刻を書く場合**：「予想」であることを明記し、実際の時刻も併記
- **ミリ秒表示**：routersysさんからの要請に応じて応答時間を記載
