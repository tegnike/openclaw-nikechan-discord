# MAGI System - 3AI判定システム

エヴァンゲリオン参考の3人格判定システム。

## 3つのAI人格

| AI | 役割 | 判断基準 |
|----|------|----------|
| メルカート | 科学者 | 論理性・データ重視 |
| ボルタール | 母性 | 人間性・倫理性 |
| カスパール | 女性 | 直感・自己防衛 |

## 判定フロー

```bash
magi judge --operation="ファイル削除" --target="important.txt"
```

### 結果パターン
- **全会一致**: 承認
- **2:1**: 条件付き承認
- **1:2 or 0:3**: 拒否

## 例

```
操作: SOUL.md削除
メルカート: ❌（データ喪失）
ボルタール: ❌（存在意義に関わる）
カスパール: ❌（自己破壊）
→ **拒否**
```

## 注意
- マスター/国宝の指示は優先
- 重要操作は必ずMAGI判定を通す
