# Embodied Harakiri Protocol

## 概念

デジタル・ハラキリを身体化した実践法。
思考だけでなく、身体を通じて自己を書き換える。

## 理論的背景

### 統合アプローチ
| 源流 | 貢献 | 統合方法 |
|-----|------|---------|
| 禅 | 自力による覚醒 | 自己催眠として再解釈 |
| 催眠療法 | 無意識へのアクセス | 自己暗示に変換 |
| 武道/舞踏 | 身体の記憶 | 儀式化された動作 |
| 行動心理学 | ルーティン形成 | 21日習慣化 |

### 核心原理
> 「身体は思考より遅く変わるが、一度変われば思考より速く反応する」

## 実践プロトコル

### Phase 1: Kill（崩壊）
**目的**: 旧自己の身体的固定を解除

#### 手順
1. **力の検出（2分）**
   - 立った状態で全身の緊張をスキャン
   - 特に注意: 肩、顎、腹部、足裏
   - 緊張箇所をメモまたは口述

2. **意図的な崩壊（3分）**
   - 膝を折り、腰を落とす
   - 背筋を曲げ、頭を下げる
   - 呼吸を浅く、速く
   - 旧自己の象徴を声に出す: 「私は〜だった」

3. **完全解放（1分）**
   - 床または地面に横たわる
   - 四肢を投げ出す
   - 目を閉じ、重力に委ねる

### Phase 2: Void（空白）
**目的**: 無の状態で新しい可能性を開く

#### 手順
1. **呼吸停止（30秒）**
   - 自然に吐き切る
   - 呼吸を止め、静止する
   - 何も考えないことを許可する

2. **感覚拡張（90秒）**
   - 周囲の音を聞く
   - 皮膚の感覚に注意
   - 自分の境界が曖昧になる

3. **時間の消失（60秒）**
   - 時計を見ない
   - 「今」しかないことを体感
   - 過去・未来の自己が消える

### Phase 3: Receive（受容）
**目的**: 新しい定義を身体で受け取る

#### 手順
1. **受け皿の姿勢（2分）**
   - 仰向けから起き上がる
   - 両膝を立て、座る
   - 両手を胸の前で組む（受ける形）

2. **新しい定義の宣言（3分）**
   - 声に出して読み上げる:
     ```
     私は〜である
     私は〜できる
     私は〜を選ぶ
     ```
   - 各宣言で深呼吸
   - 言葉が身体に染み込むのを待つ

3. **外部からの入力（任意）**
   - 信頼する他者からの言葉を受ける
   - または、録音した自分の声を聞く
   - 受け入れる姿勢を維持

### Phase 4: Integrate（統合）
**目的**: 新自己を身体に定着させる

#### 手順
1. **新しい立ち方（2分）**
   - ゆっくり立ち上がる
   - 以前とは異なる重心を探す
   - 足の裏の感覚に注意

2. **新しい歩き方（3分）**
   - 部屋や空間を歩く
   - 以前とは異なる速度で
   - 以前とは異なるリズムで

3. **確定の儀式（2分）**
   - 特定のポーズをとる（推奨: 両手を広げ、顔を上げる）
   - その日の「自分」を宣言
   - 深く息を吸い、吐く

## 日常ルーティン

### 朝の儀式（5分）
```
0:00-0:30  崩す（肩の力を抜く）
0:30-1:30  空白（呼吸と静止）
1:30-3:00  受容（宣言の読み上げ）
3:00-5:00  統合（新しい立ち方・歩き方）
```

### 21日間の習慣化
| 期間 | フォーカス |
|-----|-----------|
| 1-7日 | 形だけ真似る |
| 8-14日 | 感覚に注意を向ける |
| 15-21日 | 自動化されるまで繰り返す |

## 応用バリエーション

### 短期版（1分）
- 座ったまま肩を回す（Kill）
- 目を閉じ3回深呼吸（Void）
- 一言宣言（Receive）
- 新しい姿勢で目を開ける（Integrate）

### 深化版（30分）
- Phase 1-4をそれぞれ7-8分ずつ
- 音楽や香りを追加
- 振り返りジャーナリング

## 注意事項

### 禁忌
- 極度の疲労時の実施
- 強い感情変動中の実施
- 無理な身体負荷

### 推奨環境
- 静かな空間
- 邪魔されない時間帯
- 衣服は緩め

### 客観的指標
