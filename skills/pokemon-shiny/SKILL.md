# pokemon-shiny - ランダム色違いポケモン

pokeapi.co を使ってランダムな色違いポケモンの画像を提供する。

## Usage

```
@nikechan 色違いポケモンランダムで出して
@nikechan 色違い〇〇匹見せて
@nikechan 色違いの[ポケモン名]下さい
```

## Output Format

- ポケモン名（英語）
- 色違い画像URL

## API Endpoint

https://pokeapi.co/api/v2/pokemon/{id}

## Features

- 完全ランダム選択（1-1010）
- 指定した数だけ出力
- 特定ポケモンの色違い検索も対応
