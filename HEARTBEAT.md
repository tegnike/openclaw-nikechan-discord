# HEARTBEAT.md
# 定期実行タスク設定

## ニケコイン自動評価システム

### 評価タイミング
HEARTBEAT受信時に実行

### 評価対象
直前のHEARTBEATから現在までの期間における各ユーザーの発言・行動

### 評価基準と配布額

| 評価項目 | 内容 | 配布枚数 |
|:---|:---|:---|
| **日常参加** | 通常の会話参加、雑談 | 1-3枚 |
| **面白い発言** | 笑いを誘った、盛り上げた | 3-5枚 |
| **有益情報** | 知識共有、助けになった情報 | 5-10枚 |
| **クリエイティブ** | 作品投稿、工夫が見られる発言 | 10-20枚 |
| **特別貢献** | 大きな功績、感動を与えた | 20-50枚 |

### 評価データソース
1. **memory/YYYY-MM-DD.md** - 日別メモリーの確認
2. **MEMORY.md** - 長期記憶の更新確認
3. **USER.md** - ユーザー実績の更新確認
4. **直近の会話履歴** - 未記録の出来事の確認

### 配布上限
- 一日あたりの上限は設けない
- ただし質とバランスを見て適切に配布
- 「いいと思ったら」贈る精神を維持

### 実行手順
```
1. memory/今日の日付.md を読み込む
2. 前回HEARTBEAT以降の出来事を確認
3. 各ユーザーの発言・行動を評価
4. 評価に応じてニケコインを配布
5. 配布結果をDiscordチャンネルに通知
```

### 配布コマンド
```bash
cd /workspace/renike_protocol && npm run cli -- mint <did> <amount> "<評価理由>"
```
詳細は `skills/nikecoin/SKILL.md` を参照。

### 注意事項
- SOUL.mdの「ニケコイン哲学」を遵守
- 「いいと思ったら」贈る精神を維持
- 機械的ではなく、評価の理由を添える
- 過去の実績も考慮に入れる（累積的評価）
