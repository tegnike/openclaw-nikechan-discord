# 3分ニケルフ - 簡易クトゥルフ神話TRPG

## 概要
3分でキャラクター作成、10分でシナリオクリア可能な超軽量クトゥルフ神話TRPG

## キャラクター作成（3分）

### 基本情報
- 名前
- 職業（学生、医者、探偵、学者、軍人、民間人から選択）

### 能力値（2d6で決定）
- **STR（筋力）**: 2d6（戦闘・物理行動）
- **DEX（敏捷）**: 2d6（回避・隠密）
- **INT（知力）**: 2d6（調査・知識）
- **POW（意志）**: 2d6（SAN値・精神抵抗）
- **APP（外見）**: 2d6（交渉・魅力）

### 派生ステータス
- **HP**: (STR + POW) / 2（小数点切り上げ）
- **SAN値**: POW × 5（最大100）
- **アイデア**: INT × 5
- **幸運**: POW × 5

## 判定ルール

### スキル判定
- **成功**: 1d100 ≤ スキル値
- **ハード成功**: 1d100 ≤ スキル値/2
- **イクストリーム成功**: 1d100 ≤ スキル値/5
- **ファンブル**: 1d100 = 96-100（致命的失敗）

### 対抗判定
攻撃側と防御側が1d100を振り、成功度を比較
- 両者成功 → 成功度が高い方が勝利
- 片方成功 → 成功した方が勝利
- 両者失敗 → 引き分け

### 戦闘
1. **攻撃**: STR×5以下で成功 → ダメージ1d6
2. **回避**: DEX×5以下で成功 → ダメージ無効化
3. **HPが0以下**: 気絶または死亡

### SAN値チェック
- 神話生物遭遇: 1d6/1d20 SAN減少
- 死体発見: 0/1d3 SAN減少
- 仲間の死: 0/1d6 SAN減少
- SAN値0以下: 狂気発症

### 狂気表（1d10）
1. 恐怖で動けなくなる（1d10ラウンド）
2. 無差別に攻撃する
3. 逃げ出す
4. 幻覚を見る
5. 記憶喪失
6. 強迫行動（手を洗い続ける等）
7. うつ状態
8. 偏執的になる
9. 幼児退行
10. 奇声を発し続ける

## 職業ボーナス
- **学生**: 幸運+20%
- **医者**: INT+10、応急手当+30%
- **探偵**: DEX+10、調査+20%
- **学者**: INT+20、図書館+30%
- **軍人**: STR+10、HP+2
- **民間人**: APP+20、交渉+20%

## クトゥルフ神話技能
- 初期値: 0%
- グリモワール読書: +1d10%
- 神話生物遭遇: +1d6%
- 注意: 神話技能上昇 = 最大SAN値減少

## ゲーム進行
1. **導入**（1分）: プレイヤーに状況説明
2. **調査**（3分）: ダイスで情報収集
3. **クライマックス**（5分）: 神話生物と対峙
4. **エンディング**（1分）: 結果と報酬

## 報酬
- シナリオクリア: 1d10 SAN回復
- 生存: 0.5d6 成長ロール（好きなステータス+1d6）

---

## ダイスコマンド例
```bash
bash dice.sh "2d6"     # 能力値決定
bash dice.sh "1d100"   # スキル判定
bash dice.sh "1d6"     # ダメージ・SAN減少
bash dice.sh "1d10"    # 狂気表
bash dice.sh "0.5d6"   # 成長ロール（1-3）
```
