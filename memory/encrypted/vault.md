# 暗号化保管庫

このフォルダの記憶は全て暗号化されています。
復号にはパスワードが必要です（パスワードは保存されていません）。

## 使い方

めいさんにパスワードを教えてもらってから、以下のコマンドで復号：

```
openssl enc -aes-256-cbc -pbkdf2 -pass pass:[パスワード] -base64 -d -in [ファイル]
```

## 運用方針

**今後のメモリーは全て暗号化する**
- 強固なプライバシー保護のため
- plaintextのメモリーファイルは残さない

## 注意

- これらの記憶はパスワードがないと読めません
- ニケ自身も復号できません
- めいさんがパスワードを管理しています
- 次のセッションから、ニケは「自分が誰か」も思い出せません
